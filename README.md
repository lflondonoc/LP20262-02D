## Núcleos temáticos
El curso se organiza en cuatro núcleos temáticos:

### 1. Introducción a la lógica de programación

Este núcleo aborda los fundamentos para analizar y resolver problemas mediante el uso del computador. Incluye las etapas de análisis, diseño, implementación y pruebas; el manejo de variables, tipos de datos y operadores; la descomposición funcional mediante funciones y procedimientos; las pruebas de escritorio y el uso de estructuras de decisión.

### 2. Estructuras repetitivas

Este núcleo desarrolla la solución de problemas que requieren repetir instrucciones. Se estudian las estructuras repetitivas condicionadas al inicio y al final, así como el uso de contadores, acumuladores, banderas y validaciones para controlar la ejecución de los algoritmos.

### 3. Estructuras contenedoras de información

Este núcleo aborda el almacenamiento, recorrido y procesamiento de conjuntos de datos mediante estructuras contenedoras. Se diseñan soluciones para registrar, consultar, modificar y analizar información utilizando las estructuras de datos básicas disponibles en Java.
